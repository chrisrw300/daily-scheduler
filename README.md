# Daily Scheduler
### Description
In this program, you can:
- Write tasks within the timeframe of 9AM - 5PM
- Save/Delete tasks
- Know when your tasks are in the past, present, future
---
### This program uses the following languages:
- HTML5
- CSS
- Bootstrap
- jQuery
- Moment.js
